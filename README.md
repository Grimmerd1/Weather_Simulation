# Weather Simulation

IoT-based Weather Simulation System using ESP32, MQTT, Unity, and Meta Quest.

## Features
- Real-time sensor monitoring
- MQTT communication
- Day/Night simulation
- Rain effect
- Fog effect
- Storm simulation

## Technologies
- Unity
- C#
- ESP32
- MQTT
- DHT11
- BMP180
- Meta Quest
